翻了一下博客目录，发现不知不觉已经积累了几十篇文章。简单整理一下这个博客都写了些啥，也方便自己以后回顾。

## 碎片信息管理

最早折腾的是各种笔记和速记软件。从最开始的微信收藏，到后来专门研究了 Cubox、Flomo 这些卡片笔记工具。最后的结论是 Cubox 的 Chrome 插件体验不错，但登录体验实在让人头疼，Flomo 则太强调自己的理念。结论是与其依赖这些服务，不如自己用 NAS 搭建个自托管的方案。

## 嵌入式开发

树莓派和 Pico 是经常折腾的对象。用 Pico 驱动过电子墨水段码屏，期间被 GPT-4o 震撼到了——它居然能把我完全不懂的字节位操作讲得明明白白。之前还玩过树莓派刷机、装系统、刷 Arch Linux，以及用树莓派 TimeMachine 做私有云备份。

## 建站与部署

博客用的是 Hugo，部署过程踩过不少坑。先是从 Hexo 迁移过来，又研究了 GitHub Actions 自动部署到阿里云 OSS 的方案。期间还解决了 Hugo 模板修改、Disqus 评论替换成 Waline、阅读数统计、静态博客添加评论系统等一系列问题。

## 容器与虚拟化

Docker 相关文章占了不少篇幅。从最基础的镜像构建、容器网络配置，到给 Docker 容器配置中文环境，再到用容器运行 Streamlit 应用。中间还踩过 VSFTPD 的坑。

## 实用技巧

杂七杂八的记录还很多：SSL 证书手动更新、Git 常用命令、Mac 的 Homebrew 环境变量配置、外网访问群晖 NAS、群晖上搭建 SVN 服务器、npm 代理搭建并支持 SSL 访问等等。算是半个知识库了。

## 杂项

还写了些比较随性的东西，比如新 Mac 开箱、投影仪连接 NAS 资源、一些 Linux 命令笔记之类的。
