## 开源的速记软件—Memos
在上一篇[评测各种速记软件](https://md.winotmk.com/posts/250319-%E7%A2%8E%E7%89%87%E4%BF%A1%E6%81%AF%E7%AE%A1%E7%90%86%E9%80%9F%E8%AE%B0app/#flomo)的文章中我们找到了很多商业/免费开源的方案，最终锁定了几款最好用的：
Flomo就是一款好用的多平台速记软件，然而做任务得会员后还要订阅付费实在是让人望而却步

那么有没有此类软件的开源版本呢？

有！

非常推荐Memos：
![2025-04-30-10-38-32](http://pictures.winotmk.com/250430-%E5%BC%80%E6%BA%90%E6%9B%BF%E4%BB%A3%E4%BB%98%E8%B4%B9flomo%E7%9A%84%E6%9C%80%E4%BD%B3%E9%80%89%E6%8B%A9/2025-04-30-10-38-32_8a7f7453.png)

和Flomo界面几乎一摸一样有木有
此项目由docker实现了一个笔记服务内核与一个webui访问，另外还找了好几个配合该项目内核使用的客户端软件，可以几乎完美平替Flomo的所有功能

## Memos本体安装
https://github.com/usememos/memos
### docker方便部署
只需一行：
```
docker run -d --name memos -p 5230:5230 -v ~/.memos/:/var/opt/memos neosmemo/memos:stable
```
即可运行起memos服务和webui端
用`127.0.0.1:5230`可访问

### 套https反代
这部分在这篇文章里专门说过：https://md.winotmk.com/posts/250324-%E8%BD%BB%E6%9D%BE%E6%90%AD%E5%BB%BAnpm%E4%BB%A3%E7%90%86%E5%B9%B6%E6%94%AF%E6%8C%81ssl%E8%AE%BF%E9%97%AE/

### memos-desktop
https://github.com/xudaolong/memos-desktop
此项目维护似乎不如上面一个项目，但写出来备用，和上面一个项目几乎实现同样的功能，

使用自定义的服务器需要改个字段，见：https://github.com/xudaolong/memos-desktop/issues/4
```
打开Memos目录访问Memos\resources\app\dist\main
改main.js文件，然后搜索内容：http://localhost:${e}
将它改为你网页访问的URL，前面的e部分是端口，套过https的自然是443端口
```
